Pesquisa

O que é React?

React.js é uma biblioteca JavaScript que foi elaborada com o intuito de implementar, em aplicações web, interfaces de usuário (UI) mais eficientes e interativas, através da utilização de componentes flexíveis e reutilizáveis, conhecidos como React Components.

Como surgiu?

A biblioteca foi criada em 2011 por Jordan Walke, um engenheiro do Facebook. Posteriormente, em 2013, o Facebook decidiu transformar o React.js em uma ferramenta open source. Este foi o primeiro passo para a tecnologia atingir o nível de popularidade atual, pois possibilitou a criação de uma grande comunidade de desenvolvedores individuais, que colaboram refinando ainda mais o projeto.

Como implementar?(Pré-requisitos)

De acordo com o própro site do React, para implementar a tecnologia você necessita de navegadores e/ou dispositivos com suporte aos objetos Map e Set do Javasscript. Caso contrário, faz-se necessário a incluir um polyfill global no bundle de sua aplicação, tal como o pacote core-js.

Também é possível adicionar o React a apenas uma parte do site, com poucas linhas de código. Para isto, é necessário adicionar um container ao HTML com o id de sua aplicação referenciada em seu Componente React, criar seu Componente e adicionar as Tags Script necessárias para adicionar o React ao site. Para mais informações e tutorial completo, clique aqui.

Por que utilizar?

  • Liberdade

    Por se tratar de uma biblioteca em JS, e não um framework, o desenvolvedor, ao escolher trabalhar com React, ganha a liberdade para escolher outras ferramentas que queira utilizar em seu projeto.

  • Suporte

    Além da gigantesca comunidade de devs do mundo inteiro contribuindo para melhorar as implementações da biblioteca, a tecnologia React é amplamente utilizada pelo Facebook, o que aumenta ainda mais o suporte para a plataforma.

  • Mercado de trabalho

    O fato de ser muito bem apoiada, tanto pela comunidade como pelas grandes corporações, faz do React.js a biblioteca mais famosa na área do desenvolvimento web. Sendo assim, é um prato cheio para os desenvolvedores, novos ou experientes, que buscam por uma tecnologia para se especializar, tendo em vista a grande demanda por profissionais qualificado

  • Flexibilidade

    Caso ainda não tenha se convencido, o ecossistema da tecnologia acompanhou sua crescente popularidade, hoje em dia é possível criar soluções que vão desde aplicativos mobile, com React Native, até aplicações Desktop com Electron, e ainda a possibilidade de criar aplicações em realidade virtual, utilizando React VR. Ou seja, você, literalmente, só precisa aprender uma vez, para ter acesso a um mundo inteiro de possibilidades de implementações em React, tá esperando o quê?
